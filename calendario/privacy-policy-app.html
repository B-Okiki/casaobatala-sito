2. Dati Raccolti

L'applicazione Calendario Yorùbá NON raccoglie alcun dato personale.

L'app funziona interamente in locale sul tuo dispositivo. Nello specifico:

  • Non richiede registrazione o login
  • Non raccoglie nome, email o altri dati identificativi
  • Non utilizza cookie di tracciamento
  • Non integra servizi di analytics (Google Analytics, Facebook Pixel, ecc.)
  • Non condivide alcun dato con terze parti
  • Tutti i calcoli del calendario avvengono localmente sul dispositivo

3. Notifiche Push

L'app può richiedere il permesso di inviarti notifiche giornaliere con il giorno Yorùbá corrente.

  • Il permesso è facoltativo e puoi rifiutarlo
  • Le notifiche sono generate localmente dal tuo dispositivo
  • Nessun dato viene inviato a server esterni
  • Puoi disattivare le notifiche in qualsiasi momento dalle impostazioni del dispositivo

4. Archiviazione Locale

L'app utilizza tecnologie standard del browser per funzionare offline:

  • Service Worker: permette all'app di funzionare senza connessione internet
  • Cache locale: memorizza i file dell'app sul tuo dispositivo

Questi dati sono tecnici, non personali, e rimangono esclusivamente sul tuo dispositivo.

5. Diritti dell'Utente

Poiché non raccogliamo dati personali, non esistono dati da:

  • Accedere
  • Rettificare
  • Cancellare
  • Portare (portabilità)

Se desideri rimuovere completamente l'app e i suoi dati locali, puoi:

  • Disinstallare l'app dal dispositivo
  • Cancellare i dati del sito dalle impostazioni del browser

6. Sicurezza

L'app è distribuita tramite connessione sicura HTTPS. Non essendoci trasmissione di dati personali, non esistono rischi legati alla sicurezza dei tuoi dati.
